献礼版（先行版）介绍：
本书是文革史简明教程的先行版，也是毛主席130周年诞辰的献礼版。之后会不定期
发布修订版，下册以及完整版也会在未来写出后发布。


请读者注意：
本书虽然力求史实准确和史论合于马列毛主义，但限于水平所限，难免会有偏差。
目前来看，导言某些段落很可能还是有问题的，所以请在阅读时多思考，自行判断
论述的对错。如果本书造成了某个新谬论的流行，那就违背我们写作的本意了。


本书的读法：
注意黑体字部分；
如果有不了解的人物、集团或概念可以试试查找索引；
善用大事年表；
导言如果读不懂可以先跳过读正文，再回头读导言。
